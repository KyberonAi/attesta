## Channels

- bugs/features/docs: GitHub Issues
- code contributions: Pull Requests
- vulnerabilities: GitHub Security Advisory (private)

## Response Targets

- issue triage: 5 business days
- PR first review/update: 5 business days
- security acknowledgment: 3 business days

## Supported OSS Scope

- core Python/TypeScript SDK behavior
- public docs
- OSS no-code connectors in `packages/`

## Not Included

- guaranteed enterprise SLA response
- bespoke production incident handling
- custom integration implementation on demand
